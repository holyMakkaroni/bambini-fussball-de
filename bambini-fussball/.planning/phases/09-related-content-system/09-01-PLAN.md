# Phase 9 Plan 1: Related Articles System

## Objective

Create a related articles component that shows other articles from the same category, improving internal linking and user engagement.

## Tasks

### Task 1: Create RelatedArticles Component
**Type:** auto

Create `src/components/RelatedArticles.astro` with:
- Props: category, currentArticleId, maxArticles (default 3)
- Filter articles from same category, exclude current
- Sort by pubDate descending
- Display in responsive 3-column grid
- German heading: "Weitere Artikel fuer [category]"

### Task 2: Integrate into ArticleLayout
**Type:** auto

Update `src/layouts/ArticleLayout.astro`:
- Import RelatedArticles component
- Add articleId to Props interface
- Place component after article content slot

### Task 3: Update Category Pages
**Type:** auto

Update all three `[id].astro` pages to pass articleId:
- src/pages/eltern/[id].astro
- src/pages/trainer/[id].astro
- src/pages/vereine/[id].astro

## Verification

```bash
npm run build
```

All pages should build successfully with related articles sections.

## Success Criteria

- [ ] RelatedArticles component created
- [ ] Component integrated into ArticleLayout
- [ ] All [id].astro pages pass articleId
- [ ] Build passes without errors
- [ ] Related articles appear on article pages
